Girija's Albums (Angular & Java stack)
Personal website for sharing my Clicks
AngularTypeScriptAngular MaterialJava8SpringBoot2MySQLDocker

This project is a Progressive Web App. It allows you to see a list of albums contiaining images. You can view the images inside an album and also add a caption to them, if you would like to. The API is developed using Spring Boot 2. You can view the visited albums/images even offline. Try adding it to the home screen on your phone to get a real feel of a PWA. It uses Material design and Angular Material library. Displays Responsive images and Webp format on supported browsers to reduce the network traffic and provide a faster experience.
